# 고지자기 이동 탐색기

지구 고지자기 데이터의 판 구조 운동과 겉보기 편극 이동(APW) 경로를 3D 글로브로 보여주는 인터랙티브 웹 앱입니다. 사용자는 지질 시대를 슬라이더로 탐색하고, 판 회전과 APW만을 분리해 보며, 오일러 극과 지자기 극 궤적의 차이를 배울 수 있습니다.

## 주요 기능
- **타임라인 슬라이더:** 0–120Ma 구간을 단계적으로 이동하며 재구성을 확인합니다.
- **시나리오 전환:** 판 운동 vs APW 전용 모드를 토글해 회전과 극 이동 효과를 비교합니다.
- **학습용 콜아웃:** 오일러 극, APW, 해석 방법을 설명하는 인라인 안내를 제공합니다.
- **반응형 캔버스:** 전체 화면 WebGL 글로브와 고정형 범례·컨트롤을 제공합니다.

## 프로젝트 구조
- `index.html` — UI 레이아웃, 스타일, 컨트롤 패널과 글로브 캔버스 DOM.
- `app.js` — Three.js 기반 글로브 렌더링, 인터랙션 핸들러, 데이터 연결.
- `data/` — 시각화에 사용되는 GeoJSON 및 부가 데이터 파일.
- `testfile.py` — Git 테스트용 간단한 파이썬 스크립트.

## 시작하기
1. WebGL을 지원하는 최신 브라우저에서 `index.html`을 엽니다.
2. 타임라인 슬라이더로 시간을 이동하고 시나리오 드롭다운으로 두 모드를 비교합니다.
3. 콜아웃에 마우스를 올리거나 포커스해 핵심 지구물리 개념을 살펴봅니다.

## 개발 메모
- 글로브는 Three.js를 사용합니다. `app.js`에서 CDN을 참조한다면 네트워크 연결이 필요합니다.
- `data/` 디렉터리를 옮길 경우 `app.js`의 데이터 경로를 함께 수정하세요.
- 별도의 빌드 단계 없이 정적 페이지로 실행됩니다.
